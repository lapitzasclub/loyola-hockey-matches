# .copilot

Memoria interna de trabajo para este proyecto.

## Propósito

Esta carpeta guarda contexto operativo y técnico del proyecto para poder retomarlo rápido y mantener continuidad entre sesiones.

## Archivos

- `project-memory.md`: estado actual, arquitectura, decisiones, deuda técnica y próximos pasos.
- `worklog.md`: registro breve y cronológico de cambios, hallazgos y decisiones.
- `next-steps.md`: lista viva de tareas priorizadas.

## Mantenimiento

Actualizar esta carpeta cuando cambie alguno de estos puntos:

- arquitectura o stack
- dependencias
- decisiones técnicas
- bugs importantes detectados
- tareas prioritarias
- riesgos o limitaciones conocidas

## Notas

- Está pensada como memoria de trabajo, no como documentación pública para usuarios.
- Debe mantenerse breve, precisa y útil.
